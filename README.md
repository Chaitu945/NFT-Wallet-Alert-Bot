# 🚀 NFT Wallet Alert Bot

A Python-based Telegram bot that monitors EVM wallet activity and sends instant alerts whenever a target wallet purchases an NFT.

## ✨ Features

- 🔔 Telegram notifications
- 🖼 NFT purchase monitoring
- 🌐 EVM wallet tracking
- 🔐 Environment variable support
- 📦 Modular project structure
- ⚡ Built with Python 3.13

## 🛠 Tech Stack

- Python 3.13
- Telegram Bot API
- Alchemy API
- Web3.py
- Requests
- python-dotenv

## 📁 Project Structure

```
NFTAlertBot/
│
├── .env
├── config.py
```
